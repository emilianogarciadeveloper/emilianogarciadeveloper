## How I work

**Fast, stable, maintainable — in that order.**

I treat every website like a production system: it has to load fast, stay up when traffic and pressure hit, and be easy for your team to maintain long after I hand it off. No bloated page builders, no plugin soup, no surprises — just clean work that holds up.

I've spent 8+ years building and fixing real projects for clients across the US, Spain, Canada and Australia, working fully remote.
